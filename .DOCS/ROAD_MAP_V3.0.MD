# 🦅 CONDOR — ROADMAP ESTRATÉGICO Y TÉCNICO

---

# 🎯 VISIÓN

Condor es una herramienta de analytics diseñada para personas no técnicas, enfocada en interpretar datos y sugerir acciones concretas.

---

# 🧠 POSICIONAMIENTO

"Analytics for non-analytics people"

Condor no muestra datos.
Condor explica qué significan.

---

# 🚀 FASE 1 — MVP (0 → VALIDACIÓN)

## 🎯 Objetivo
- lanzar rápido
- validar con usuarios reales
- crear valor con datos manuales

---

## 🧱 Features

- Upload CSV (LinkedIn, Instagram, etc.)
- Parsing automático
- Dashboard básico
- Métricas clave:
  - impresiones
  - likes
  - comentarios
  - engagement rate

---

## 🧠 Feature diferencial

### Insight automático (reglas simples)

Ejemplos:

- "Este post tuvo 2x engagement vs promedio"
- "Publicaste menos que la semana anterior"
- "Los posts con imagen rinden mejor"

---

## ⚙️ Stack

- Next.js
- Tailwind
- Supabase
- PapaParse (CSV parsing)

---

## 💸 Costos

- $0 (free stack)

---
